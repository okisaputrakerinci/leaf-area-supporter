Leaf Area Supporter

Aplikasi ini digunakan untuk proses anotasi dan masking gambar pada daun sebagai bagian dari pra-pemrosesan dalam pengolahan citra digital.

Output utama dari aplikasi ini berupa file .yaml yang dapat digunakan dalam pelatihan model atau pengelolaan dataset anotasi.

Tujuan Aplikasi

  • Membuat anotasi objek daun secara manual
  • Melakukan masking objek dari citra asli
  • Menghasilkan file YAML sebagai format label
