# LeafLabeler (GUI)
- Jalankan dev: python run.py (butuh Python 3.11+, Windows/Linux/Mac)
- Hotkeys: [1]=FG, [2]=BG, [3]=Wand, [R]=Quick Select, [C]=Clear, [S]=Save, Shift+S=Save&Next, A/D=Prev/Next, Ctrl+0=Fit, [ / ]=Brush -, +
- Export dataset YOLO: menu panel kanan → "⇪ Export YOLO Dataset"
- Export mask biner: "⬜ Sel=Putih" / "⬛ Sel=Hitam" (file: <nama>_mask.png)
